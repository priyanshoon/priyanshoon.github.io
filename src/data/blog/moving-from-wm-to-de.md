Not so long ago, I have been using Hyprland as my window manager. It was great, it has so many functionality 
out of the box. After a while my interest goes towards niri, and man, niri is way too good. I like it's 
infinite scrolling functionality too much that I don't want to go to any other wm or de.

But, yeah I know "BUT" always hurts. Niri is good and all, it has some drawback, like some software 
does not run at all (the software written in java 😑) which I don't like it. I need applications
like burpsuite, ghidra to test software as I am moving towards security more and more. I tried GNOME, but tbh
it's too bloated for me. So I decided to try out KDE (also I use nixos btw). And boy I was impressed, it's too
modern according to my needs yet much lighter than GNOME. Now I can watch anime while being afk thanks to 
softwares like kde connect. 

I don't know what else to write now, that's all I wanted to share btw. PEACE!
